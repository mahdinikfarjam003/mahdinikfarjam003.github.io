## About This Blog

This blog focuses on security research, detection engineering, and threat hunting content.

Topics include:

- Windows Lateral Movement techniques
- DCOM / COM exploitation research
- SIEM detection engineering (Splunk, etc.)
- MITRE ATT&CK analysis
- Proof of Concept (PoC) development

The goal of this project is to document real-world attacker techniques and translate them into actionable detections for SOC environments.
